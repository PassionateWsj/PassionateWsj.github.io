Launcher3壁纸修改

  1. 将资源图片放在Launcher3\WallpaperPicker\res\drawable目录下，并改好缩略图，命名规则为资源图后加_small
  2. 在Launcher3\WallpaperPicker\res\values-nodpi\wallpapers.xml文件中声明资源图片，可不包含后缀名

根据plmn获取不同主页

  1. Browser\src\com\android\browser\BrowserSettings.java文件中getHomePageFromPLMN()方法添加case
  2. 并且在setHomePage()方法中添加相应项目的判断
    if(SystemProperties.get("ro.project.customer_name").equals("Bmobile")){
            sFactoryResetUrl = getHomePageFromPLMN();
        }

修改系统默认值

  1. 修改frameworks\base\packages\SettingsProvider\res\values\defaults.xml文件中对应的值，比如：def_wifi_display_on和def_wifi_on设置为默认开启
  2. 字体的修改在\frameworks\base\core\java\android\content\res\Configuration.java文件的setToDefaults()方法中，fontScale = 1； //normal value is 1

Tips

@ALL MTK 7.0 平台 Kernel开启了device-mapper-verity (dm-verity) 功能导致userdebug版本无法remount /system为rw模式

需要 
1)执行adb disable-verity 
2) reboot
3) 重新root /remount
