2017-09 小知识点整理

发表于 2017-09-30
  1. 加强推送稳定性，及时性
    • 使用多平台推送（根据不同品牌，注册不同推送服务：小米，华为，魅族，三方 sdk ）
      • MisPush( MixPush SDK 融合多家推送平台，小米推送、魅族推送、个推等，在 MIUI 和 Flyme OS 共享系统级推送，杀死 APP 也能收到推送消息。)
      • 参考文章
    • 衍生问题
      • 推送是通过设备的唯一标识进行匹配的，如果通知注册后，后台在每个平台都对所有设备发送一次通知，是否某个品牌手机会收到多条通知

  2. 集成三方 sdk 需要获取证书指纹

    • 获取方式1

      • 终端切换到 .android 目录
      • 输入指令

    • 获取方式2 (调用脚本)

      • ./getcertificate.sh [apk]
